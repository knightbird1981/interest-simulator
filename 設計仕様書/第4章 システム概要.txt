# 第4章 システム概要

## 4.1 システム概要

積立運用シミュレーターは、積立投資における将来資産の推移をシミュレーションするWebアプリケーションである。

利用者は積立金額や運用利回り、運用期間などの条件を入力することで、将来の資産推移をグラフおよび数値で確認できる。

本システムは、初心者でも直感的に利用できる操作性を重視し、入力内容の変更結果を即座に反映することで、様々な運用条件を比較・検討できることを目的とする。

---

## 4.2 システムの目的

本システムの目的は、積立運用の結果を分かりやすく可視化し、利用者が将来の資産形成をシミュレーションできる環境を提供することである。

主な目的は以下のとおり。

- 積立投資の将来資産を試算する
- 運用条件の違いによる資産推移を比較する
- グラフによる視覚的な理解を支援する
- 投資初心者でも迷わず利用できるUIを提供する
- ブラウザのみで利用できるシンプルなシステムとする

---

## 4.3 システム構成

本システムは、フロントエンドを中心としたWebアプリケーションとして構築する。

基本構成は以下のとおり。

| 区分 | 内容 |
|------|------|
| クライアント | Webブラウザ |
| フロントエンド | HTML / CSS / JavaScript |
| グラフ表示 | Chart.js |
| データ保存 | なし（初版） |
| サーバー | なし（静的サイト） |
| 配布方法 | GitHub Pages |

初版（Ver.1.0）はサーバーレス構成とし、利用者の入力内容はブラウザ内で処理を行う。

---

## 4.4 利用者

本システムの想定利用者は以下とする。

| 利用者 | 内容 |
|---------|------|
| 一般利用者 | 積立運用を試算したい個人 |
| 投資初心者 | 積立投資の仕組みを学びたい利用者 |
| 資産形成検討者 | 条件を比較しながら将来資産を検討したい利用者 |

利用者登録やログイン機能は実装しない。

---

## 4.5 動作環境

本システムは主要ブラウザでの利用を前提とする。

| 項目 | 内容 |
|------|------|
| OS | Windows、macOS、Android、iOS |
| ブラウザ | Google Chrome、Microsoft Edge、Safari |
| インターネット接続 | 必須（GitHub Pages利用時） |
| JavaScript | 有効であること |

最新版の主要ブラウザで正常動作することを前提とする。

---

開発環境は以下を基本とする。

| 項目 | 内容 |
|------|------|
| ソース管理 | GitHub |
| エディタ | Visual Studio Code |

本システムで使用する技術は以下とする。

| 分類 | 使用技術 |
|------|----------|
| マークアップ | HTML5 |

| プログラミング | JavaScript（ES6以降） |

| リポジトリ | GitHub |
